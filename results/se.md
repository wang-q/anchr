## se: template

* template

```shell script
WORKING_DIR=${HOME}/data/anchr
BASE_NAME=se

cd ${WORKING_DIR}/${BASE_NAME}

rm *.sh
anchr template \
    --se \
    --genome 4641652 \
    --parallel 24 \
    --xmx 80g \
    \
    --trim "--dedupe --tile --cutoff 5 --cutk 31" \
    --qual "25 30" \
    --len "60" \
    --filter "adapter artifact" \
    \
    --quorum \
    --cov "40 80" \
    --statp 2 \
    --redo \
    \
    --extend

```
